Для данных вероятностей составим систему уравнений в конечных разностях [4,5,6]. 
Для упрощения выражений введем индикатор:
\begin{equation*}
	E_a^b = 
	\begin{cases}
		1, &{a=b}\\
		0, &{a \neq b}
	\end{cases}
\end{equation*}
\begin{equation*}
	\overline{E}_{a}^b = 1 - E_a^b
\end{equation*}
\begin{adjustwidth}{-40pt}{100pt}
